# Ywis Beauty - Professional Beauty Salon Website

Professionelle, moderne und SEO-optimierte Website für das Beauty Homestudio **Ywis Beauty** in Cloppenburg, Deutschland.

## 📋 Projektübersicht

Eine vollständig funktionsfähige Website für ein Beauty Homestudio mit den Dienstleistungen:
- 💅 Nageldesign
- ✨ Wimpern Lifting (Lash Lifting)

**Besonderheit:** Terminvereinbarung ausschließlich über Social Media (WhatsApp, Instagram, TikTok)

## 🚀 Inbetriebnahme & Setup

### 1. Platzhalter ersetzen

Alle Platzhalter müssen ersetzt werden:

```
PLACEHOLDER_OWNER        → Name der Betreiberin
PLACEHOLDER_STRASSE      → Straße und Hausnummer
PLACEHOLDER_WHATSAPP     → WhatsApp Nummer (z.B. +49 123 456789)
PLACEHOLDER_INSTAGRAM    → Instagram Username (ohne @)
PLACEHOLDER_TIKTOK       → TikTok Username (ohne @)
PLACEHOLDER_EMAIL        → Email (optional)
PLACEHOLDER_VAT_ID       → Umsatzsteuer-ID (optional)
PLACEHOLDER_GOOGLE_VERIFICATION → Google Search Console Code
```

### 2. Bilder einfügen

Erforderliche Bilder:

```
/images/
├── logo.png                    # Logo (mindestens 200x200px)
├── banner.jpg                  # Hero Banner (1920x1080px oder größer)
├── about-hero.jpg              # Über-Mich Bild (800x600px)
├── nageldesign-hero.jpg        # Nageldesign Hero (800x600px)
├── wimpern-lifting-hero.jpg    # Wimpern Hero (800x600px)
├── og-image.jpg                # Social Media Preview (1200x630px)
├── favicon.png                 # Favicon (any size, wird reskaliert)
├── apple-touch-icon.png        # Apple Touch Icon (180x180px)
└── /gallery
    ├── nageldesign-1.jpg bis nageldesign-6.jpg
    ├── wimpern-1.jpg bis wimpern-3.jpg
    └── ... weitere Bilder
```

### 3. Hosting & Domain

- Website ist vollständig statisch (keine Server-Logik nötig)

- Kann zu github.com hochgeladen werden (kostenlos mit GitHub Pages)
